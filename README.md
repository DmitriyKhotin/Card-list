# Card-list
Simple SPA with cards list

## Installation

**1. Download the project**

```bash
git clone https://github.com/Dmitriymgtu/Card-list.git
```
**2. Go to the application folder**

**3. Run application**

```bash
npm dev
```

or

```bash
yarn dev
```
